The s.l-m (slim) api!


This is a simple api that allows you to shorten your links and store plaintext data similar to pastebin. This originally was a small project I wrote up on and off over a couple weeks while learning full stack web development. I returned to it to rewrite the backend into a more robust and scalable api using Redis and the V langage.

It's a very simple service by design, this is just a frontend for the underlying api. Only necessary data is stored, dont believe me? Full Source Code

Clicking on the square below switches between short links and plaintext mode. The return link is automatically copied into the clipboard.

The /api/b route is a new addition for advanced users. It allows you to store binary data in the database with a fixed 48 hour expiry. More information in the docs.
